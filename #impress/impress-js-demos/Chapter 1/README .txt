Example Source Codes for the Chapter 1 
=======================================


Folder Structure
=======================================

css           - Custom styles file for the example presentation
js            - impress.js library file
chapter1.html - Contains the HTML code for the presentation
README.md     - All the licensing,versioning information about the framework


How to Use
=======================================

Open the chapter1.html file in a web browser which supports impress.js and
use arrow keys or spacebar to run the presentation.
